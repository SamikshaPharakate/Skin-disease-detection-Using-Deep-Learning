
**Skin Disease Detection using Deep Learning**

This project implements a skin disease classification system using Deep Learning and Transfer Learning techniques. Multiple models, including **CNN, MobileNetV2, and ResNet50**, are trained and evaluated on a skin disease image dataset to compare their performance and identify the most effective architecture.

#### Features

* Image preprocessing and normalization
* Skin disease image classification
* CNN, MobileNetV2, and ResNet50 implementation
* Accuracy and loss visualization
* Model performance comparison
* Prediction on unseen test images

#### Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Google Colab

#### Objective

To develop an automated skin disease detection system and analyze the effectiveness of different deep learning models for medical image classification.
